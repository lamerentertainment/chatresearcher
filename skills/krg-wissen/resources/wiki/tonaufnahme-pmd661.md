## Zusammenfassung

Diese Anleitung beschreibt das Verschieben von Tonaufnahmen vom Gerät PMD661 auf das Laufwerk des Kriminalgerichts Luzern.

---

## Vorgehen

### 1. Vorbereitung

- Tonaufnahmegerät PMD661 bereithalten
- PC am Arbeitsplatz starten
- USB-Kabel oder Kartenleser bereithalten

### 2. Datei vom Gerät übertragen

#### Variante A: Mit USB-Kabel
1. PMD661 mit USB-Kabel am PC anschliessen
2. Gerät einschalten
3. Im Windows Explorer das Gerät als Laufwerk öffnen
4. Gewünschte Audiodatei(en) auswählen
5. Auf den Desktop oder in einen temporären Ordner kopieren

#### Variante B: Mit SD-Karte
1. SD-Karte aus dem PMD661 entnehmen
2. SD-Karte in den Kartenleser am PC einstecken
3. Im Windows Explorer die SD-Karte öffnen
4. Gewünschte Audiodatei(en) auswählen
5. Auf den Desktop oder in einen temporären Ordner kopieren [Quelle: Datei_von_Tonaufnahmegerät_zu_Laufwerk_verschieben_PMD661.docx]

### 3. Datei auf Laufwerk verschieben

1. Windows Explorer öffnen (Windows + E)
2. Zum Zielverzeichnis navigieren, z.B.:
   - G:\KRG\Dokumente\Tonaufnahmen\[Jahr]
   - G:\KRG\Dokumente\[Fallnummer]\Audio
3. Datei(en) vom Desktop in das Zielverzeichnis verschieben (Drag & Drop)
4. Datei sinnvoll benennen, z.B.:
   - [Fallnummer]_Verhandlung_[Datum].mp3
   - [Fallnummer]_Befragung_[Name]_[Datum].mp3 [Quelle: Datei_von_Tonaufnahmegerät_zu_Laufwerk_verschieben_PMD661.docx]

### 4. Aufräumen

- Temporäre Dateien auf dem Desktop löschen
- SD-Karte wieder ins Gerät einlegen (falls verwendet)
- PMD661 ausschalten und verstauen [Quelle: Datei_von_Tonaufnahmegerät_zu_Laufwerk_verschieben_PMD661.docx]
