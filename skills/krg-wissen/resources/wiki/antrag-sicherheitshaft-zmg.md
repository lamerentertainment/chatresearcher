## Zusammenfassung

Dieses Dokument ist eine Vorlage für einen Antrag auf Anordnung von Sicherheitshaft an das Zwangsmassnahmengericht (ZMG).

---

## Antragstellung

### Zuständigkeit

Der Antrag auf Anordnung von Sicherheitshaft wird beim Zwangsmassnahmengericht (ZMG) gestellt. [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

### Antragsberechtigung

- Staatsanwaltschaft
- Kriminalgericht [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

## Antraginhalt

### 1. Betroffene Person

- Name, Vorname
- Geburtsdatum
- Nationalität
- Wohnort [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

### 2. Straftatbestände

Auflistung der relevanten Straftatbestände mit Artikelangaben. [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

### 3. Haftgründe

Darlegung der Haftgründe:
- Fluchtgefahr (Art. 221 Abs. 1 lit. a StPO)
- Verdunkelungsgefahr (Art. 221 Abs. 1 lit. b StPO)
- Fortsetzungsgefahr (Art. 221 Abs. 1 lit. c StPO)
- Ausführungsgefahr (Art. 221 Abs. 2 StPO) [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

### 4. Sachverhalt

Kurze Darstellung des relevanten Sachverhalts. [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

### 5. Antrag

Konkreter Antrag auf Anordnung der Sicherheitshaft. [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

## Verfahren

### Einreichung

- Schriftlich beim ZMG
- Per E-Mail oder A-Post
- Bei Dringlichkeit: Telefonische Voranmeldung [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]

### Entscheid

Das ZMG entscheidet innert kurzer Frist über den Antrag. [Quelle: Antrag auf Anordnung von Sicherheitshaft an ZMG.docx]
