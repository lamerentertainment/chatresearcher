## Zusammenfassung

Diese Anleitung beschreibt das Abspielen von Audio- und Videodaten im Kontext von Gerichtsverhandlungen.

---

## Vorgehen

### 1. Daten lokalisieren

- Datei im Laufwerk G suchen
- Typische Pfade:
  - G:\KRG\Dokumente\Tonaufnahmen\[Jahr]
  - G:\KRG\Dokumente\Videoaufnahmen\[Jahr]
  - G:\KRG\Dokumente\[Fallnummer]\Medien [Quelle: Abspielen von Daten.docx]

### 2. Abspielprogramm wählen

#### Für Audiodateien
- Windows Media Player
- VLC Media Player
- Spezialisierte Transkriptionssoftware [Quelle: Abspielen von Daten.docx]

#### Für Videodateien
- VLC Media Player (empfohlen)
- Windows Media Player
- Tribuna-interner Player [Quelle: Abspielen von Daten.docx]

### 3. Datei öffnen

- Programm starten
- Datei über "Öffnen" auswählen
- Oder: Doppelklick auf Datei [Quelle: Abspielen von Daten.docx]

### 4. Wiedergabe steuern

- Start/Stop
- Lautstärke
- Vorspulen/Zurückspulen
- Bei Bedarf: Untertitel/Sprachkanal wählen [Quelle: Abspielen von Daten.docx]

## Tipps

### Qualität prüfen

- Bei schlechter Qualität: IT-Support kontaktieren
- Ggf. Originaldatei beiziehen
- Backup-Kopien verwenden [Quelle: Abspielen von Daten.docx]

### Für Verhandlungen

- Rechtzeitig vor Verhandlung testen
- Reserve-Abspielgerät bereithalten
- Lautstärke im Saal vorher einstellen [Quelle: Abspielen von Daten.docx]
